Aide

Mode d'emploi :
1. Cliquez sur un nœud (ou une question de la liste à gauche, ou une réponse de son menu déplié) pour le modifier dans le panneau de droite.
2. Une fois vos modifications faites dans le panneau de droite, cliquez sur « Valider » pour les appliquer à l'arbre.
3. La forme de l'arbre est calculée automatiquement : vous ne pouvez pas déplacer un nœud à la main, elle se réorganise toute seule dès que vous ajoutez/supprimez un élément.
4. Question vs Étape : Une Question peut avoir plusieurs réponses (branches). Une Étape permet soit une suite unique (flèche sans texte), soit de se diviser en plusieurs branches nommées (comme une question).
5. Si un nœud est sélectionné (en surbrillance), cliquer sur « + » crée directement le lien avec ce nœud. Sinon, utilisez « Relier deux nœuds » : cliquez le parent puis l'enfant.
6. Utilisez la barre de recherche pour retrouver un nœud par mot-clé : l'arbre se centre sur lui et le sélectionne pour modification immédiate.
7. Ctrl+Z pour annuler, Ctrl+Maj+Z pour rétablir.
8. Touche Suppr (ou Backspace hors champ texte) pour supprimer le nœud ou la branche sélectionné(e).

Mise à jour GitHub :
1. Cliquez sur « Vers GitHub ».
2. Allez dans le dépôt GitHub de l'Atlas.
3. Supprimez l'ancien arbre_dynamique.json.
4. Ajoutez le fichier téléchargé.
5. Vérifiez qu'il s'appelle exactement arbre_dynamique.json.
6. Cliquez sur « Commit changes ».
