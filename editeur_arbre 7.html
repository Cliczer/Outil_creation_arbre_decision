Aide

Mode d'emploi :
1. Cliquez sur un nœud (ou une question de la liste à gauche, ou une réponse de son menu déplié) pour le modifier dans le panneau de droite.
2. Une fois vos modifications faites dans le panneau de droite, cliquez sur « Valider » pour les appliquer à l'arbre.
3. La forme de l'arbre est calculée automatiquement : vous ne pouvez pas déplacer un nœud à la main, elle se réorganise toute seule dès que vous ajoutez/supprimez une question, une décision ou une branche.
4. Pour relier deux nœuds : bouton « Relier deux nœuds », puis cliquez le parent puis l'enfant.
5. Pour vous déplacer : cliquez-glissez sur le fond, utilisez les barres de défilement, ou la molette de la souris au-dessus de l'arbre pour zoomer/dézoomer.
6. Utilisez la barre de recherche pour retrouver un nœud par mot-clé : l'arbre se centre sur lui et le sélectionne pour modification immédiate.
7. Ctrl+Z pour annuler, Ctrl+Maj+Z pour rétablir.

Mise à jour GitHub :
1. Cliquez sur « Vers GitHub ».
2. Allez dans le dépôt GitHub de l'Atlas.
3. Supprimez l'ancien arbre_dynamique.json.
4. Ajoutez le fichier téléchargé.
5. Vérifiez qu'il s'appelle exactement arbre_dynamique.json.
6. Cliquez sur « Commit changes ».
